browser-mario
=============

Play Mario in your browser

Goals:
1.  is to use a socket connection to allow more than one player
2.  Divs will be breakable if set to
3.  Divs will take up space
	- Mario will interact with Divs by:
	- Preventing movement if trying to walk into one
	- Act as a floor
	- If not set to breakable, a div will stop from further movement upwards
	- Spawn objects if set to(mushrooms, coins, vines, etc.)
	- Act as an object in code with properties and methods
.
